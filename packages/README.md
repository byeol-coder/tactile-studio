# Tactile Packages

Batch A adds two prepare-only packages:

- `@dot/tactile-contract`: shared tactile grid primitives and validation helpers.
- `@dot/tactile-core`: a pure JavaScript port of the image conversion engine, DTMS/DTM codec helpers, parity tests, and a Node import check.

This batch is package-only core/contract preparation. It does not add app wiring, hardware integration, workflow changes, or visual identity work.
