# DotPadSDK-3.0.0

This directory vendors `DotPadSDK-3.0.0` for the Studio DotPad BLE/USB connection layer.

The SDK file is redistributed here only so the Studio build can bundle the browser-side DotPad transport. Confirm the exact upstream license and redistribution wording with Dot Inc before treating this vendored copy as final.

Browser transport requirements:

- Chromium-based browser
- HTTPS or localhost
- User gesture before opening the Web Bluetooth or Web Serial chooser
