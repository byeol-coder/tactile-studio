# Dot Canvas toolbar icons

Figma의 `Icon Components — Dot Canvas Toolbar` 프레임에서 추출한 인라인 SVG입니다.

- 아이콘: 23종
- 상태: `default`, `selected`
- 전체 파일: 46개
- 크기: 53 × 53
- 구조: `icons/<name>/<state>.svg`
- 색상: Figma 원본의 CSS custom property와 fallback 값을 유지

HTML에서는 SVG 파일 내용을 그대로 복사해 인라인으로 사용하거나, 빌드 도구의 SVG raw import 기능으로 불러오면 됩니다.

노드 ID와 한글 이름의 전체 대응표는 `manifest.json`을 참고하세요.
